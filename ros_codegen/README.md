# ROS codegen for matlab SOP repo

*Peter Chang*

## objective
This repo is set to learn the ins and outs of devloping an controller that
is best used for ROS/ROS2 systems, later at the high level, deployed on systems
such as Jetson Orin, Jeston Nano, or further at a lower level, such as
Respberry pi, and STM32. The goal is to test and code-gen from matlab.

## target
This repo is set to make different `control01.mlx` for description on different 
control algorightm, starting from encoder read-in, feedback signal filtering,
later to the state-space controller desing, and use a MATLAB live script to
test and check it's performance.

### possibilitys
- code gen with .m file for direct c/c++ for embedded system
- code gen with .slx file for in-direct ROS2 system at high level
